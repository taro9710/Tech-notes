## ¿Qué es Linux (o más bien GNU/Linux)? ¿Qué es una Distro?
### Sistema operativo
Un sistema operativo (OS operating system) es el conjunto del software que permite usar la computadora. Esto incluiría:
- El **kernel**
- Herramientas para interactuar con el sistema (como terminales, gestores de archivo, etc)
- Librerías
- Interfaces gráficas
### Kernel
El kernel o núcleo es el corazón del sistema operativo. Es un programa que siempre se está ejecutando en segundo plano y tiene control total sobre todo el sistema. Se encarga de: 
- Gestionar la memeoria RAM
- Controla los dispositivos de hardware (CPU, teclado, etc)
- Coordina la ejecución entre software y hardware
### Linux Kernel 
El Kernel de linux (llamado simplemente Linux), desarrollado por Linus Torvalds desde 1991 es un kernel basado de código abierto (FOSS free and opens-source software) basado en el sistema operativo de Unix System. Y que fué adoptado por el GNU operating system, que fue creado para ser una opción "free" (gratis/libre) para reemplazar a Unix.
### GNU/Linux
Es el sistema operativo y basado en el **kernel de linux** y muchas de las herramientas del proyecto **GNU** creado por Richard Stallman. GNU desarrolló herramientas esenciales como compiladores (`gcc`), terminales (`bash`), bibliotecas(`glibc`), etc. Y junto a kernel de Linux ambos formaron un sistema operativo funcional y libre, por eso aunque coloquialmente se le dice "Linux" al sistema operativo se recomienda fuertemente el uso del nombre GNU/Linux.
