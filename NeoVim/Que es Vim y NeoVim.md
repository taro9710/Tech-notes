# QUE ES VIM

    - Es un editor de texto.
    - Funciona mediante modos.
    - Es perfecto para programar.
    - Funciona desde la terminal.
    - Es muy ligero y consume muy pocos recursos.
    - Es extremadabmente configurable.
    - Tiene una comunidad muy grande.
    - Funciona para cualquier lenguaje de Programación.
    - Es multiplataforma.
    - Vim es un editor diferente al resto, y tiene una curva de aprendizaje que puede ser eleevada porque está pensado para utilizarse con el teclado y sin ratón.

# QUE ES NEOVIM. Vim + LUA + LSP + GRAN COMUNIDAD

    - También es un editor de texto.
    - Es un fork de Vim. Por lo que todo lo que funciona en vim, también funciona en NeoVim.
    - Una versión más moderna y extensible de Vim.
    - Podemos utilizar LUA parar configurar, crear temas y plugins.
    - Comunidad cada vez más grande.
    - Tiene soporte nativo para LSP (Language Server Protocol). (Osea lo que noos permite tener cosas como autocompletados, saltar a definición, diagnóstico de errores, etc.)


# TIPS Y CONSEJOS ANTES DE EMPEZAR.

    - Mente abierta, porque muchas cosas se hacen diferentes.
    - Aprender a escribir correctamente en el teclado.
    - Vim no es un editor que se puede aprender en un día. Dominarlo puede tomar años.
    - En las primeras semanas no esperar a ser productivo o rápido.Quizás no empezar a proyectos con Vim todavía.
    - Vim no va a reemplazar el editor de texto todavía.
    - Aprender todos los atajoss y comandos poco a poco.
    - Hacer los 128 ejercicios para aprender atajos.
    - Utilizar hojas de ayuda para memorizar las teclas y los comandos.
